# Интеграция с Яндекс Картами (Laravel + Vue)

Небольшое приложение на Laravel 12 + Vue (Inertia), которое реализует интеграцию с Яндекс Картами по ТЗ:

- Авторизация (логин/пароль) на базе Laravel Breeze.
- Страница настроек с сохранением ссылки на карточку компании в Яндекс.Картах.
- Страница отзывов с отображением рейтинга, общего количества отзывов, списка отзывов, сортировкой по новизне и постраничной навигацией (на текущий момент на тестовых данных).

## Стек

- PHP 8.5+
- Laravel 12
- Laravel Breeze (Inertia + Vue 3)
- SQLite (по умолчанию)
- Node.js + Vite (сборка фронтенда)

## Установка и запуск локально

```bash
# Клонировать репозиторий
git clone <URL_ВАШЕГО_РЕПОЗИТОРИЯ> imtera-test
cd imtera-test

# Установить PHP-зависимости
composer install

# Установить JS-зависимости
npm install

# Скопировать .env и сгенерировать ключ
cp .env.example .env
php artisan key:generate

# Запустить миграции (SQLite по умолчанию)
php artisan migrate

# Запустить dev-сервер Laravel
php artisan serve

# В отдельном окне терминала запустить Vite
npm run dev
